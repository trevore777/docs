GuitarSongBook

Platform: iOS (SwiftUI)

Purpose: Worship song library app with editable playlists, live play view, chords/lyrics bundled as JSON.

Key features / Frameworks

  • Songs and playlists stored as local JSON files in app bundle
  • Edit/add songs & playlists (local storage)

App Store Connect Notes

  • Category: Music (primary); Lifestyle or Reference possible secondary
  • Age rating: 4+
  • Privacy: On-device only; “No data collected”
  • Copyright: Only distribute songs you have rights for; provide copyright details
Privacy Policy: All musical data stored locally; edits remain on user device.
