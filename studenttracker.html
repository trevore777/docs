StudentTracker

Platform: iOS (SwiftUI)

Purpose: Teacher tool for logging student wellbeing, behaviour, lunchtime reflections, export/email records.

Key features / Frameworks

  • Local data storage (student names & comments)
  • Email export option (uses device’s Mail compose)

App Store Connect Notes

  • Category: Education or Productivity
  • Age rating: 4+ (staff app, mild content)
  • Privacy: Data stored only on-device; "No data collected"
  • Distribution: Prefer organisation Apple ID and unlisted distribution for schools
Privacy Policy: Holds student data locally only; no sync/export to external services.
