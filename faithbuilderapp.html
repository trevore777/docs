FaithBuilderApp

Platform: iOS (SwiftUI)

Purpose: Bible study helper for Christian-school students; offers topic-based studies, scriptures, and notes.

Key features / Frameworks

  • Uses offline/local data for topics & verses
  • No analytics or authentication configured

App Store Connect Notes

  • Category: Education or Reference
  • Age rating: 4+ or 9+ (no user content, you control all content)
  • Privacy: Data stored only on-device; "No data collected"
  • TestFlight: Defined description, name, subtitle combos, and icon ideas
Privacy Policy: No accounts, no external data; all Bible study topics/notes stored locally.
