Bible Hub App

Platform: iOS (SwiftUI with Firebase)

Purpose: Store and search Bible studies by topic, fetch verses (KJV API), sync via Firestore.

Key features / Frameworks

  • Firebase Auth (email/password) & Firestore for syncing studies
  • KJV Bible API for verses
